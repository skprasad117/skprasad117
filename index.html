About

Data infrastructure meets intelligent systems

I'm a Data Engineer and AI/ML practitioner working at the intersection of scalable data infrastructure and intelligent systems. My work revolves around designing robust data architectures that power analytics, machine learning, and real-time decision-making across cloud environments.

On the intelligence side, I design and deploy ML models, build RAG-based agents, and leverage GenAI tools to create solutions that go beyond traditional analytics.

Whether it's data lakehouse architecture, deep learning workflows, or crafting end-to-end GenAI applications, I bring a software engineering mindset — writing clean, testable code with a focus on reliability, scalability, and maintainability.
